Kariyer Hedefim

Mezuniyetim sonrası ilgimi çeken diğer alanlara göre daha çok üzerinde durduğum Frontend Web Developing ve Full Stack Java developing alanında kariyerimi ilerletmek. Frontend Web Geliştirme ve Java hakkında internet üzerinden aldığım çeşitli eğitimler ile konu hakkında teorik ve pratik bilgiye sahibim, hedefim bu alanlarda çalışan ekip arkadaşlarıyla görev almak, kendimi ön yüz(Frontend) web ve Java geliştirme alanları öncelikli olmak üzere yazılım alanında geliştirmek ve farkındalık kazanmak.

Eğitimim

Atatürk Üniversitesi Yönetim Bilişim Sistemleri Bölümünden mezun oldum. bitirme projem Blockchain tabanlı sistemlerin geçmişden günümüze kullanımı, altyapısı, geleceği hakkında yapmış olduğum teknik dokümantasyondur.
